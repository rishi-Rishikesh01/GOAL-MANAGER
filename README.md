# 🎯 Goal Manager App

A cross-platform mobile application built with **React Native** and **Expo** for tracking daily goals, tasks, and productivity.

## 📱 Features

- ✅ Add, edit, and delete daily goals
- 📊 Track progress and productivity
- 🔔 Task reminders and notifications
- 📱 Works on both Android and iOS
- 💾 Persistent local storage

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React Native | Cross-platform mobile framework |
| Expo | Development and build tooling |
| JavaScript | Core programming language |
| AsyncStorage | Local data persistence |
| React Navigation | In-app navigation |

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or above)
- Expo CLI
- Expo Go app on your phone

### Installation

```bash
# Clone the repository
git clone https://github.com/rishi-Rishikesh01/GOAL-MANAGER.git

# Navigate to project directory
cd GOAL-MANAGER

# Install dependencies
npm install

# Start the development server
npx expo start
```

### Running the App
1. Install **Expo Go** on your Android or iOS device
2. Scan the QR code shown in the terminal
3. The app will load on your device
